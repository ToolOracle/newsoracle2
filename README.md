## Tools (8)

| Tool | Description |
|------|-------------|
| `news_headlines` | Top headlines by category and language. Categories: business, tech, world, finan |
| `company_news` | All recent news about a specific company. Optionally compare with competitors. |
| `topic_feed` | News feed for any custom topic or keyword. |
| `multi_topic_feed` | Monitor up to 5 topics simultaneously. Returns latest 3 articles per topic. |
| `industry_pulse` | Industry-specific news pulse. Industries: e-commerce, immobilien, finanzen, tech |
